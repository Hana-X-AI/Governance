## What is N8N?

N8N is a workflow automation platform that helps you:
- Connect different apps and services together
- Automate repetitive tasks
- Move data between systems
- Build business processes without coding

**Popular Use Cases**:
- Email automation
- Data synchronization
- Report generation
- Approval workflows
- Notification systems

## Important Notes

**Security**:
- Always use HTTPS URLs (https://...)
- Your credentials are encrypted
- Don't share your password

**Browser Compatibility**:
- Works best with: Chrome, Firefox, Edge, Safari
- Use latest browser version
- Enable JavaScript

**Network Access**:
- N8N is only accessible from internal network (hx.dev.local domain)
- Not accessible from outside the company network
