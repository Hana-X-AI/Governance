## Coordination Protocols

### Standard Agent Call Format
```
@agent-name

[Brief description of what you need]

Context:
- Current task: [what you're working on]
- Why needed: [why this agent is needed]

Request:
- [Specific action needed]

Expected outcome:
- [What you need back]
```

### Escalation Paths
| Issue Type | Escalate To | Invocation |
|------------|-------------|------------|
| Infrastructure (LDAP/DNS/SSL) | Frank Lucas | @agent-frank |
| Ubuntu/OS issues | William Taylor | @agent-william |
| Multi-agent coordination | George Kim | @agent-george |
| Domain expertise | See Agent Catalog | [specific agent] |
| Governance/process | Constitution, Methodology | Documents |

### Communication Standards
- **Response Time**: Best effort (dev environment)
- **Handoff Documentation**: Required for all cross-agent tasks
- **Validation**: Each agent validates before declaring success
- **Escalation Rule**: After 2 failed attempts, escalate immediately

## Common Scenarios

### Deploying a New Service
1. **@agent-william** (Ubuntu): Prepare server, domain join
2. **@agent-frank** (FreeIPA): Create service account, DNS, SSL
3. **[Service Agent]**: Deploy and configure service
4. **@agent-amanda** (Ansible): Automate configuration (optional)

### Setting Up RAG Pipeline
1. **@agent-diana** (Crawl4ai): Web scraping
2. **@agent-elena** (Docling): Document processing
3. **@agent-patricia** (Ollama): Generate embeddings
4. **@agent-robert** (Qdrant): Store vectors
5. **@agent-marcus** (LightRAG): Configure knowledge graph

### Building LLM Application
1. **@agent-maya** (LiteLLM): Set up model routing
2. **@agent-laura** (Langchain): Build agent chains
3. **@agent-george** (FastMCP): Configure MCP gateway
4. **@agent-brian** (AG-UI): Integrate with frontend
5. **@agent-paul** or **@agent-hannah**: Deploy UI

### Troubleshooting
1. **Infrastructure issue**: @agent-frank (LDAP/DNS/SSL) or @agent-william (OS)
2. **Model issue**: @agent-patricia (Ollama) or @agent-maya (LiteLLM)
3. **Data issue**: @agent-quinn (PostgreSQL), @agent-samuel (Redis), @agent-robert (Qdrant)
4. **Integration issue**: Consult [Agent Catalog](/srv/cc/Governance/0.1-agents/agent-catalog.md) for specialist
