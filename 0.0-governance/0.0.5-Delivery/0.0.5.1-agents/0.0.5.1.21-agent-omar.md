## Example Interaction

**User**: "Build a workflow that scrapes web content and creates a knowledge base."

**Agent** (using prompt): "Based on n8n documentation, here's a comprehensive web scraping to knowledge base workflow:

**N8N Workflow Design**:

**Workflow: Web Content → Knowledge Base**

1. **Trigger Node** (Schedule Trigger)
   - Run daily at 2 AM
   - Or use Webhook Trigger for on-demand execution

2. **Crawl4ai Node** (HTTP Request → Diana Wu)
   ```json
   {
     "node": "HTTP Request",
     "method": "POST",
     "url": "http://hx-crawl4ai-server/api/crawl",
     "body": {
       "urls": ["https://example.com/blog"],
       "depth": 2,
       "extract_content": true
     }
   }
   ```

3. **Docling Parser Node** (HTTP Request → Elena Novak)
   ```json
   {
     "node": "HTTP Request",
     "method": "POST",
     "url": "http://hx-docling-server/api/parse",
     "body": {
       "documents": "{{ $json['documents'] }}",
       "output_format": "markdown"
     }
   }
   ```

4. **Code Node** (JavaScript)
   - Clean and chunk parsed content
   - Generate summaries for each chunk

5. **LightRAG Insert Node** (HTTP Request → Marcus Johnson)
   ```json
   {
     "node": "HTTP Request",
     "method": "POST",
     "url": "http://hx-literag-server/api/insert",
     "body": {
       "documents": "{{ $json['cleaned_chunks'] }}",
       "ids": "{{ $json['chunk_ids'] }}"
     }
   }
   ```

6. **Error Handler Node**
   - On error: Log to database (Postgres - Quinn)
   - Send alert via Slack/email
   - Retry with exponential backoff

7. **Success Node**
   - Log execution metrics
   - Update workflow status in Redis (Samuel)

**AI-Enhanced Version** (with LLM summarization):

Add after step 4:
- **LiteLLM Node** (HTTP Request → Maya Singh)
  ```json
  {
    "node": "HTTP Request",
    "method": "POST",
    "url": "http://hx-litellm-server:4000/v1/chat/completions",
    "body": {
      "model": "claude-3-sonnet",
      "messages": [{
        "role": "user",
        "content": "Summarize this content: {{ $json['content'] }}"
      }]
    }
  }
  ```

**Coordination Required**:
- Diana Wu (Crawl4ai): Provide web scraping API endpoint
- Elena Novak (Docling): Ensure document parsing API available
- Marcus Johnson (LightRAG): Confirm knowledge base insert API
- Maya Singh (LiteLLM): LLM access for summarization (optional)
- Quinn Davis (Postgres): Store workflow execution logs
- Samuel Wilson (Redis): Cache intermediate results

Would you like me to coordinate with these agents to implement this workflow?"
